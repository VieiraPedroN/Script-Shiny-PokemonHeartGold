# Script Shiny Hunt - Eevee (Pokémon HeartGold)

Script em Lua para automatizar e agilizar a **Shiny Hunt do Eevee** em **Pokémon HeartGold** utilizando o emulador **DraStic para Android**.

O script executa automaticamente as etapas necessárias após conversar com o Bill, permitindo verificar rapidamente se o Eevee recebido é shiny.

---

# Pré-requisitos

Antes de utilizar o script, certifique-se de que:

## 1. Posicionamento

Seu personagem deve estar posicionado **de frente para o Bill**.

> <img width="1356" height="610" alt="Posicionamento" src="https://github.com/user-attachments/assets/cda00da7-9f1b-4ff0-8570-2fc0ac7d6d1a" />

## 2. Salvar o jogo

Salve o jogo utilizando o sistema de save do próprio Pokémon:

**Menu → Save**

Isso garante que, após cada reset, seja possível reiniciar o processo corretamente.

## 3. Slot do Eevee

O espaço destinado ao Eevee deve ser o **último slot disponível da equipe**.

# Como utilizar

1. Abra o DraStic.
2. Selecione **Load New Game**.
3. Escolha a ROM de **Pokémon HeartGold**.
4. Aguarde o carregamento do save.
5. Execute o script Lua.

Após iniciado, o processo será totalmente automatizado.

---

# Funcionamento do Script

O script executa as seguintes etapas:

1. Inicia o jogo.
2. Carrega o save.
3. Conversa com o Bill.
4. Recebe o Eevee.
5. Abre a tela de Status do Pokémon.

Assim, você pode verificar rapidamente se o Eevee recebido é shiny.

---

# Reset para nova tentativa

Caso o Eevee não seja shiny, utilize o atalho de reset do DraStic:

```text
L + R + Select + Start
```

O jogo será reiniciado e o script poderá ser executado novamente.

---

# Dica

Recomenda-se utilizar o recurso **Fast Forward** do DraStic para acelerar as tentativas e reduzir o tempo de cada ciclo.
